- Instalação no MacOS: https://pplware.sapo.pt/tutoriais/node-js-aprenda-instalar-no-macos/

#### Começando um projeto Node.js

```
    npm  init 
```

```js
console.log('Hello node');

const a = 1;
const b = 10;

console.log(a + b);
```

```
> node .\index.js
Hello node
11
```

```json
{
  "name": "01-npm",
```

```
> npm run dev
```
